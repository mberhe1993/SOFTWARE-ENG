To manage Products please first make sure to read the instruction and key points provided below. And click the appropriate menu item located on the nav bar above. Problem Statement: Assume you have been hired by a local Grocery Store named, Golden Arch Market, and you are required to design and develop a simple web-based software solution for them, which will be part of their Inventory Management system. Specifically, the system will be used in managing data about the Products they have in stock. They want you to implement a basic web application for this purpose. Especially important to the Store Manager, is the data that provides information about the Total Value (in dollars and cents) of all the Products currently in stock. The Total Value is the sum total of the value of all the Products in stock. It is calculated by summing the value of each of the Products. And the value of each Product is calculated by multiplying its Unit Price by the Quantity in stock.
